# TEST API POKEMON

A simple project to test APIs REST with Mocha, SuperTest e Chai

API Example on test - (http://pokeapi.co)

## Getting Started

#### Dependencies

* Mocha
* SuperTest
* Chai

#### Setup
* git clone https://github.com/WELDISSON/api-pokemon.git
* npm install

## Running the tests

* npm test (in root directory)
* mocha -R spec file path
